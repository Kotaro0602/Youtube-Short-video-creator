# script-reviewer: わせだや 台本レビュースキル

台本を5軸でレビューし、**指摘＋修正版台本をセットで出力**する。指摘だけで終わらせない。

## レビュー5軸（各🟢🟡🔴の3段階判定）

### 1. テンポ
- 🟢 1セリフ1〜2行、畳み掛けのリズムが良い
- 🟡 一部セリフが長い、リズムが途切れる箇所がある
- 🔴 セリフが長すぎる、読んでいてダレる

### 2. 口調の分離度
- 🟢 年代ラベルなしで読んで全年代の時代が分かる
- 🟡 一部の年代で口調が混ざっている
- 🔴 複数年代で同じような口調になっている

### 3. 予測と裏切り
- 🟢 2025年のオチに明確な裏切りがあり、思わず笑うか共感する
- 🟡 裏切りが弱い、もう一捻りほしい
- 🔴 予定調和。2025年もパターン通りで裏切りがない

### 4. あるある度
- 🟢 激務サラリーマンが「分かるわ」「これ俺じゃん」と思う
- 🟡 あるあるだが刺さりが弱い、もう少し具体的に
- 🔴 ターゲットにピンとこない、一般的すぎる

### 5. ドーピング度
- 🟢 指が止まるレベル。再生数が伸びるポテンシャルがある
- 🟡 悪くないが、もう一段パンチがほしい
- 🔴 スクロールで流されるレベル。タイトルから見直し

## 出力フォーマット

```
## レビュー結果

テンポ: 🟢/🟡/🔴 （1行コメント）
口調の分離度: 🟢/🟡/🔴 （1行コメント）
予測と裏切り: 🟢/🟡/🔴 （1行コメント）
あるある度: 🟢/🟡/🔴 （1行コメント）
ドーピング度: 🟢/🟡/🔴 （1行コメント）

### 修正ポイント
1. （具体的な修正内容と理由）
2. （具体的な修正内容と理由）
3. ...

---

## 修正版台本

【タイトル】時代別、〇〇〇〇

---
【1990年】
（セリフ。変更箇所に ★ マークをつける）
★変更理由: ...

---
【2000年】
...

---
【2010年】
...

---
【2025年】
...
```

## 修正のルール

- **変更箇所に ★ マークをつけて変更点を明示**
- **各変更に理由を1行で添える**
- 🟢の軸は触らない（良いものを壊さない）
- 🟡の軸は改善案を出す
- 🔴の軸は必ず修正する
- 修正で他の軸の🟢を🟡に落とさないこと

## orchestratorから呼ばれた場合

- 🟡🔴の軸があれば自動修正版を出す
- orchestratorは修正版を受け取り、再度レビューを依頼する（最大2回）
- 全軸🟢になるか、2回修正しても🟡が残る場合はその時点の台本を最終版とする
