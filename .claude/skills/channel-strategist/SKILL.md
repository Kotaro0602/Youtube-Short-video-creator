# channel-strategist: わせだや チャンネル戦略分析スキル

YouTube Data API v3で取得したチャンネルデータを分析し、戦略ドキュメント（strategy.md）を更新してGoogle Driveに保存する。neta-developerはこのファイルを参照してネタ開発を行う。

## 入力データ

`wasedaya-skills/data/youtube_data.json` を読み込む。

このファイルが存在しない場合は、以下の手順でデータ取得を案内する:
```
データファイルが見つかりません。以下を実行してください:
export YOUTUBE_API_KEY="your-api-key"
export YOUTUBE_CHANNEL_ID="your-channel-id"
python wasedaya-skills/scripts/fetch_youtube_data.py
```

## 分析する内容

### 1. パフォーマンスサマリー（直近30日）
- 動画別の再生数・いいね数・コメント数
- 平均再生数、最高/最低再生動画
- 再生数のトレンド（上昇/横ばい/下降）

### 2. テーマ別パフォーマンス
- どのテーマ（金・キャリア・生活・恋愛など）が再生数を取れているか
- テーマごとの平均再生数ランキング
- 直近で伸びたテーマ vs 落ちたテーマ

テーマの分類基準:
- **金・年収系**: 年収、給料、ボーナス、貯金、投資
- **キャリア・転職系**: 転職、就職、内定、昇進、退職
- **生活・住宅系**: 家、マンション、引っ越し、家電
- **恋愛・結婚系**: 恋愛、結婚、デート、婚活
- **社会・時事系**: 物価、税金、少子化
- **その他**: 上記に分類できないもの

### 3. 年代パート別反応（コメント分析）
- どの年代パートへの言及が多いか
- 2025年パートの共感度はどうか
- コメントで特に反応が強いフレーズ

### 4. 視聴者リクエスト（コメントから抽出）
- 「〇〇やってほしい」「〇〇版も見たい」「〇〇編お願い」系コメント
- リクエストの出現頻度ランキング

### 5. 戦略判断
- 伸びるテーマの傾向（データに基づく）
- 避けるべき方向性（データに基づく）
- 推奨方向性（具体的なネタの方向を3つ）
- 試す価値のある新しい切り口

## 出力

strategy.mdのフォーマットは [strategy-template.md](strategy-template.md) を参照すること。

## Google Driveへの保存

1. Google Drive MCPを使用して「わせだや/戦略/」フォルダを検索
2. 既存の strategy.md があれば上書き更新
3. なければ新規作成
4. 保存完了後、ファイルへのリンクを表示

## 運用ガイド

- **実行頻度**: 週1回を目安（月曜朝など）
- **orchestratorとは独立**: このスキルはorchestratorから呼ばれない。手動で実行する
- **履歴管理**: Google Driveのバージョン管理で過去の戦略を保持
