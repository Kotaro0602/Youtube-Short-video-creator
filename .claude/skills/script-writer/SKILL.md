# script-writer: わせだや 台本執筆スキル

リサーチ素材をわせだやトーンの台本に変換する。

## 台本サンプル

必ず [examples.md](examples.md) を読み、サンプル台本のテンポ・口調・構造を完全に理解してから書き始めること。

## 3原則

1. **短いカットでテンポよく** - 1セリフ=1〜2行、1年代3〜5セリフ
2. **分かりやすい横軸を提示** - タイトルで即座に構造が分かる
3. **予測させて、少しだけ裏切る** - 1990〜2010でパターン学習 → 2025で裏切り

## セリフルール

- **1セリフ40文字以内**目安
- **各年代の口調が完全に別人**（年代ラベルなしでも時代が分かるレベル）
- **固有名詞は1年代に1〜2個**（多すぎると説明くさい）
- **1セリフに情報は1つだけ**（詰め込まない）
- **読点「、、」で間を演出**（句点「。」は使わない傾向）

## 口調テンプレート（厳守）

### 1990年
- 一人称: 「俺」
- 語尾: 「〜だぜ」「〜だろ？」「〜してやるよ」「〜じゃねえか」
- トーン: でかい声、自信満々、金持ちアピール
- NGワード: 「ですかね」「かなぁ」（他年代の口調が混ざるのは絶対NG）

### 2000年
- 一人称: 「私」
- 語尾: 「〜ですかねぇ」「〜んですけどね」「ありがたいことに」
- トーン: 控えめ、腰低い、卑屈、感謝
- NGワード: 「俺」「だぜ」（1990年の口調が混ざるのは絶対NG）

### 2010年
- 一人称: 「うち」「僕」
- 語尾: 「〜ですよね」「〜じゃないですか」「いや〜」
- トーン: フラット、余裕、ちょっとドヤ
- NGワード: 「ですかねぇ」「やな」（他年代の口調が混ざるのは絶対NG）

### 2025年
- 一人称: 省略多め
- 語尾: 「〜かなぁ」「〜やな」「〜やし」
- トーン: ドライ、独り言、達観、感情薄め
- NGワード: 「だぜ」「ですかねぇ」（他年代の口調が混ざるのは絶対NG）

## 2025年オチ3パターン（いずれかを使う）

- **A: リアル共感オチ** - 分かりすぎる…（例: 年収1000万でも家買えない）
- **B: ドライオチ** - 感情なさすぎて逆に面白い（例: 内定もらっても「もっと行ける」）
- **C: 逆転オチ** - 一見良さそうで実はきつい（例: 子育て支援いいけど家は…）

## 台本フォーマット

```
【タイトル】時代別、〇〇〇〇

---
【1990年】
（セリフ3〜5行）

---
【2000年】
（セリフ3〜5行）

---
【2010年】
（セリフ3〜5行）

---
【2025年】
（セリフ3〜5行。ここがオチ）
```

## episode-orchestratorから呼ばれた場合

エピソード分析シートとリサーチシートの両方が提供される。以下のルールに従う:

### エピソード配置年代のセリフ
1. ユーザーの実体験エピソードを **その年代の口調テンプレートに完全変換** して使う
2. エピソードの「面白い核」（おもしろポイント）は必ず残す。ディテールは口調に合わせて変えてよい
3. エピソードが長い場合は3〜5セリフに凝縮する（1セリフ40文字ルールは厳守）
4. エピソードをそのまま使うのではなく、「あるある」として昇華させる

### 他年代のセリフ
- エピソードと同じテーマで、各年代の「あるある」を書く（通常の台本と同じ）
- エピソード配置年代との **対比** が際立つように書く
- 1990→2010で「パターン学習」させ、エピソード年代で「予測と裏切り」を決める

### 構成の調整
- エピソード配置年代が2025年の場合: 通常通り最後に配置（オチ）
- エピソード配置年代が2025年以外の場合:
  - そのエピソード年代のセリフを特に印象的にする
  - 2025年は別途オチを作る（エピソードのテーマに沿った2025年版の裏切り）
