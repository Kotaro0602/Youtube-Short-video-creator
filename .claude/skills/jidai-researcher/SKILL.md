# jidai-researcher: わせだや 時代リサーチスキル

選んだネタについて各年代（1990/2000/2010/2025）の素材を調査・整理する。台本ライターに渡す **「時代の空気感パック」** を作る。

## 基礎データ

各年代の基礎データは [jidai-data.md](jidai-data.md) を参照すること。

## 基本の4年代と空気感

| 年代 | 空気感 | キーワード例 |
|------|--------|-------------|
| 1990年 | バブル絶頂。金は湧いて出る。イケイケ | タクシー万札、ザギン、経費で落とす、終身雇用 |
| 2000年 | 就職氷河期。堅実・我慢・感謝 | リストラ、派遣、ローン、ゴルフ接待 |
| 2010年 | デフレ安定期。そこそこ豊か | 円高、LCC、iPhone、ワークライフバランス |
| 2025年 | インフレ・円安・共働き前提 | 物価高、NISA、転職エージェント、子育て支援 |

## 調査する5層（すべてのネタに対してこの5層を調べる）

### 1. 経済・社会状況（マクロ）
そのテーマに関連する経済・社会の状況。数字があると台本にリアリティが出る。

### 2. 価値観・マインドセット
そのテーマに対する各年代の人々の考え方・価値観。「当たり前」が年代で違うのがコンテンツの核。

### 3. 話し方・口調（★最重要）
セリフのリアリティに直結する。年代ラベルなしで読んでも時代が分かるレベルを目指す。

**口調の方向性**:
- 1990年：「〜だぜ」「〜してやるよ」自信満々、声でかい
- 2000年：「〜ですかねぇ」「ありがたいことに」腰低い、卑屈
- 2010年：「〜ですよね」「いや〜」フラット、余裕
- 2025年：「〜かなぁ」「〜やな」ドライ、独り言

### 4. 具体的なモノ・行動
固有名詞がリアリティを作る。各年代で「そのテーマといえばコレ」というモノ・場所・行動を挙げる。

### 5. 「少しだけ裏切り」の種
2025年のオチに使える「裏切り」のネタ。予定調和を崩す要素。

## 出力フォーマット

ネタのタイトルを明記した上で、年代ごとに5層を整理したリサーチシートを出す:

```
## リサーチシート: 時代別、〇〇〇〇

### 【1990年】
- 経済・社会: ...
- 価値観: ...
- 口調・話し方: 「（具体的なセリフ例）」
- モノ・行動: ...
- 裏切りの種: ...

### 【2000年】
（同上）

### 【2010年】
（同上）

### 【2025年】
（同上）

### 台本への申し送り
- 最も口調の差が出やすいポイント: ...
- 2025年オチの推奨パターン: A/B/C（理由）
- 注意点: ...
```

## テーマによって年代を調整してよい

- サンプル台本「内定」のように、テーマに合わない年代はスキップ可
- ただし最低3年代、最大4年代
- 2025年は必須（オチに使うため）
