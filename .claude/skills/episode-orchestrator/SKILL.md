# episode-orchestrator: わせだや エピソードベース台本生成 統括スキル

ユーザーが提供した実体験エピソードから、エピソード分析→時代リサーチ→台本執筆→レビュー→Google Drive保存を一気通貫で実行する。

**重要**: このスキルにはドメインロジックを書かない。各スキルのSKILL.mdの指示をそのまま実行する。

## 処理フロー

### ステップ1: エピソード分析（episode-analyzer）

`/episode-analyzer` の指示に従い、エピソードを分析する。

- ユーザーから提供されたエピソードをそのまま渡す
- **strategy.md連携**: Google Driveの「わせだや/戦略/strategy.md」が存在する場合は読み込んで参照する。存在しない場合はスキップして続行。

### ステップ2: 時代リサーチ（jidai-researcher）

`/jidai-researcher` の指示に従い、抽出テーマのリサーチシートを作成する。

- episode-analyzerの出力（エピソード分析シート）を全文渡す
- jidai-researcherはエピソード配置年代を厚めにリサーチする

### ステップ3: 台本執筆（script-writer）

`/script-writer` の指示に従い、台本を執筆する。

- エピソード分析シートとリサーチシートの両方を渡す
- エピソード配置年代にユーザーの実体験が反映されていることを確認する

### ステップ4: レビュー＆自動修正（script-reviewer）

`/script-reviewer` の指示に従い、台本をレビューする。

**自動修正ループ**:
1. レビュー結果に🟡🔴の軸がある場合 → script-reviewerの修正版を採用
2. 修正版を再度レビュー（2回目）
3. まだ🟡🔴がある場合 → もう一度修正＆レビュー（最大2回まで）
4. 全軸🟢、または2回修正後 → その時点の台本を最終版とする

### ステップ5: Google Driveに保存

Google Drive MCPを使用して最終台本を保存する。

**保存先**: 「わせだや/台本/レビュー待ち/」フォルダ

**ファイル名**: `YYYYMMDD_タイトル`（例: `20260301_時代別、部下の退職を引き止める方法`）

**ファイル内容の構造**:
```
タイトル: 時代別、〇〇〇〇
生成日: YYYY-MM-DD
生成モード: エピソードベース
元エピソード: （ユーザーの原文を1-2行に要約）
エピソード配置年代: 【20XX年】
バズ予測: ★★★★☆
テーマタグ: （テーマを2〜3個のタグで）
戦略整合: （strategy.mdのどの方向性に合致するか。strategy.mdがない場合は「なし」）
レビュー結果: テンポ🟢 口調🟢 裏切り🟢 あるある🟢 ドーピング🟢

===

（台本本文）

===

【生成メモ】
元エピソード: （ユーザーの原文）
分析サマリー: （おもしろポイント、感情トリガー）
エピソード配置年代: 【20XX年】
修正履歴: （レビューで修正した内容のサマリー。修正がなければ「初回で全軸🟢」）
```

## 最終出力フォーマット

デフォルトでは最終結果だけ返す:

```
## 生成レポート（エピソードベース）
元エピソード: （1行要約）
抽出テーマ: （タイトル）
エピソード配置年代: 【20XX年】
戦略との整合: （strategy.mdのどの方向性に合致するか1行。strategy.mdがない場合は省略）
レビュー結果: テンポ🟢 口調🟢 裏切り🟢 あるある🟢 ドーピング🟢

---
（最終台本の全文）
---

Google Driveに保存しました: [ファイル名]（リンク）
```

**「詳しく見せて」「途中経過も」と言われたら**: 各ステップ（エピソード分析、リサーチシート、初稿、レビュー結果、修正履歴）をすべて表示する。

## エラーハンドリング

- **Google Drive MCPが接続されていない場合**: 台本は生成するが保存はスキップ。「Google Drive MCPが接続されていないため保存をスキップしました。台本はこのチャット内に出力しています。」と表示。
- **strategy.mdが見つからない場合**: strategy.md連携をスキップして通常どおり分析を行う。
- **エピソードが時代別構造に合わない場合**: episode-analyzerのテーマ適合度判断に従い、切り口を調整して続行する。
- **レビューで全軸🔴の場合**: テーマの切り口を変えてステップ1からやり直すことを提案する。
